Mobile Guide

Alles von unterwegs — Scan, Upload, Verwalten.

Login required — nur Google & GitHub.

1 — Anmelden

1
Google oder GitHub wählen
G Login oder GH Login antippen
[ SCREENSHOT: Mobile Login-Seite ]
G Login (Google) oder GH Login (GitHub) oben rechts antippen. Nach der OAuth-Freigabe wird dein Konto erstellt. Erstnutzer müssen vom Admin aktiviert werden.

2 — Dateien hochladen

2
Auf +Upload tippen, Datei wählen
Upload-Button im Client-Dashboard
[ SCREENSHOT: Mobile Upload-Button ]
Im Client-Dashboard auf +Upload tippen. .md, .json, .csv, .yaml (max 500 KB) auswählen. Der Upload geht in die Admin-Prüfung.

3 — Freigabe abwarten

3
Admin prüft und genehmigt
Status im Upload-Journal einsehbar
[ SCREENSHOT: Mobile Upload-History ]
Der Admin prüft deine Datei. Bei Genehmigung erscheint sie in deiner Library. Bei Ablehnung siehst du den Grund im Upload-Journal.

4 — Library & Journal

4
Drei Tabs: Library · History · Chat
Tabs wechseln durch Antippen
[ SCREENSHOT: Mobile Client Dashboard Tabs ]
Library: Genehmigte Dateien. Upload History: Alle Uploads mit Status. Chat: Support kontaktieren.

5 — Scan Wizard

5
Geräte-Scan für .md Konvertierung
Schritt-für-Schritt-Assistent
[ SCREENSHOT: Mobile Scan Wizard ]
Unter /scan findest du den Device Scan Wizard — führt dich durch die Konvertierung zu .md.
