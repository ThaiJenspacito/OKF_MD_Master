## What is OKF?

Open Knowledge Format (OKF) structures knowledge as YAML-frontmatter-enriched Markdown files. Each skill has a `name`, `description`, `type`, `tags`, and `version` — making it machine-readable for AI agents, RAG systems, and LLM fine-tuning.
