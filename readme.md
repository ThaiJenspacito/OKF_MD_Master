# 🗂️ OKF MD Master

An autonomous, locally-hosted knowledge bridge that transforms unstructured Markdown notes into the standardized **Google Open Knowledge Format (OKF)** using local folder-watching and the **Gemini 3.1 Flash-Lite** API. Features a real-time dark-mode management dashboard and optional automated push notifications.

---

## 🎯 Vision & Purpose

In the age of AI agents and distributed teams, **Knowledge is Liquidity**. However, most personal and professional knowledge bases (Obsidian, Notion, simple text files) suffer from a lack of strict metadata structure, making them hard for advanced AI tools or Google AI Studio sync mechanisms to parse accurately.

**OKF MD Master** solves this by acting as an autonomous, invisible infrastructure layer:
* **Frictionless Workflow:** You simply type standard Markdown notes in your favorite editor.
* **Autonomous Structuring:** The second you save, the system intercepts the file, detects context, and enriches it via specialized AI with high-grade YAML Frontmatter.
* **AI Readiness:** The resulting artifacts conform exactly to the Google Open Knowledge Format, ready to be ingested instantly as system instructions or skills by AI agents.
