## 🚀 OKF MD Master คืออะไร?

เฟรมเวิร์กโอเพนซอร์สตัวแรกที่รวม Agent อัตโนมัติ 9 ตัวเพื่อเปลี่ยนไฟล์ Markdown ของคุณให้เป็น OKF Skills ที่พร้อมใช้งานกับ AI ทันที

## ✨ คุณสมบัติหลัก

- **Zero Data Loss** — ทุกไฟล์ถูกเก็บ 4 ชุด ไม่มีอะไรถูกลบ
- **AI-Powered** — DeepSeek, Gemini, Cohere ผ่าน Smart Routing
- **P2P Network** — แชร์พลังประมวลผล รับเครดิตดาวน์โหลด
- **Offline-Ready** — ทำงานได้แม้ไม่มีเน็ต (Gemini Nano)
- **Free for 100 users** — ฟรีไม่มีค่าใช้จ่าย

## ⚡ เริ่มต้นใน 30 วินาที

```bash
git clone https://github.com/ThaiJenspacito/OKF_MD_Master.git
cd OKF_MD_Master
npm install
npm start
```

เปิด `http://localhost:5000` → **Continue without password**
