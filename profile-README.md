### About

Building the **Open Knowledge Format** ecosystem — a complete pipeline that turns any text into structured, AI-ready knowledge.

**OKF MD Master** features:
- 9 autonomous agents (Watcher, Scanner, Scout, Architect, Scheduler, Quality, Skill, GitHub Bot, Social Manager)
- Multi-platform bots: Telegram, LINE, WhatsApp, Google Chat
- P2P sync, quality scoring, credit system
- Admin approval workflow + multi-tenant
- PWA installable on mobile
