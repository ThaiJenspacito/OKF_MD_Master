# Weekly Planning Assistant

An AI agent skill for creating structured weekly plans with priorities, time blocks, and recovery periods.

## The Process

1. Collect information from the user:
   - Top 3 priorities for the week
   - Fixed appointments (meetings, deadlines)
   - Personal goals (exercise, learning, family)

2. Create a weekly plan with:
   - Daily time blocks (4-5 hours productive work)
   - Buffer times between tasks
   - One "nothing day" (e.g. Sunday)

3. Save the plan for reference

4. Ask the user: "Should I add these to your calendar?"

## Use Cases

- Freelancers managing multiple projects
- Team leads planning sprints
- Students organizing study schedules
- Anyone wanting better work-life balance
