# Daily Summary Generator

An AI agent skill for creating end-of-day summaries with completed tasks, open items, and tomorrow's plan.

## The Process

1. Collect all completed tasks for the day
2. Identify open tasks (for tomorrow)
3. Create a 3-point priority list for tomorrow
4. Ask the user: "Want this summary as email or in chat?"

## Use Cases

- Daily standup preparation
- Personal productivity tracking
- Team progress reports
- Client status updates

## Best Practices

- Run this skill at the end of each workday
- Keep summaries concise (3-5 bullet points)
- Track patterns over time for productivity insights
- Share with team members for better alignment
